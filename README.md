# Fever 个人游戏作品集网站

这是一个现代、响应式的个人作品集网站，专门展示游戏开发作品。

## 🎯 功能特点

### 1. 现代UI设计
- **游戏主题界面**：使用游戏风格的设计元素和动画
- **响应式布局**：适配手机、平板和桌面设备
- **深色/浅色主题**：支持主题切换，自动适配系统偏好
- **毛玻璃效果**：导航栏和部分元素使用毛玻璃效果

### 2. 作品展示
- **游戏卡片展示**：美观的游戏卡片，包含标题、描述和分类
- **详情模态框**：点击卡片查看游戏详细信息
- **直接游戏链接**：一键跳转到实际游戏页面
- **分类标签**：按游戏类型分类展示

### 3. 交互功能
- **平滑滚动**：页面内导航平滑滚动
- **主题切换**：点击按钮切换深色/浅色模式
- **响应式导航**：移动端汉堡菜单
- **分享功能**：支持社交媒体分享
- **返回顶部**：滚动时显示返回顶部按钮

### 4. 技术特色
- **设计令牌系统**：统一的CSS变量管理
- **组件化架构**：可复用的UI组件
- **可访问性优化**：WCAG AA标准兼容
- **性能优化**：图片懒加载、代码分割

## 🚀 如何使用

### 在线访问
网站可以直接在浏览器中打开 `index.html` 文件使用。

### 本地开发
1. 克隆或下载项目文件
2. 用浏览器打开 `index.html`
3. 如需修改，编辑对应文件即可

### 自定义配置
1. **修改游戏数据**：编辑 `js/script.js` 中的 `gamesData` 数组
2. **调整样式**：编辑 `css/style.css` 中的CSS变量
3. **更新个人信息**：编辑 `index.html` 中的相关内容

## 🎨 设计系统

### 色彩系统
- **主色**：蓝色渐变 (#38bdf8 → #0284c7)
- **强调色**：橙色 (#f59e0b)
- **中性色**：灰阶系统
- **深色模式**：自动适配系统主题

### 字体系统
- **主字体**：Inter (现代无衬线字体)
- **标题字体**：Orbitron (游戏风格字体)
- **字体比例**：基于黄金比例的响应式字号

### 间距系统
- **基础单位**：4px
- **间距比例**：4、8、12、16、24、32、48、64px

## 📱 响应式断点

- **移动端**：< 640px
- **平板端**：640px - 1024px
- **桌面端**：> 1024px

## ♿ 可访问性

- **键盘导航**：支持Tab键导航
- **屏幕阅读器**：语义化HTML结构
- **色彩对比度**：符合WCAG AA标准（4.5:1）
- **焦点管理**：清晰的焦点指示器
