| 游戏名称 | 文件名 | 建议尺寸 |

| 坦克对战游戏 | `tank-game.png` | 800×600像素 |

## 📦 上传步骤

### 方法一：直接复制文件
1. 找到你的游戏图片文件
2. 复制文件（Ctrl+C）
3. 打开文件资源管理器，导航到：
   ```
   c:/Users/17115/WorkBuddy/20260409223312/assets/
   ```
4. 粘贴文件（Ctrl+V）

### 方法二：拖放文件
1. 打开你的图片所在文件夹
2. 打开assets文件夹
3. 将图片文件拖放到assets文件夹中

## 🖼️ 图片要求

1. **格式**：PNG或JPG
2. **尺寸**：建议800×600像素或16:9比例
3. **大小**：每张图片小于1MB
4. **内容**：清晰展示游戏主题
5. **命名**：必须与上述文件名完全一致

## 🔧 图片加载失败处理

我已经在代码中添加了错误处理机制：
  - 如果图片加载失败，会自动显示一个蓝色的占位图片
  - 控制台会显示警告信息，指出哪个图片加载失败
  - 占位图片会提示"请上传图片到assets目录"

## 🚀 测试图片是否生效

完成图片上传后：
1. 双击打开 `index.html` 文件
2. 滚动到"我的网页游戏"部分
3. 查看4个游戏卡片是否显示了正确的图片
4. 点击任意游戏卡片，在模态框中查看大图

## 💡 备选方案

## ❓ 常见问题

**Q: 图片显示为蓝色占位图怎么办？**
A: 检查文件名是否完全匹配，图片是否在assets目录中。

**Q: 如何更改图片？**
A: 用新的图片文件替换assets目录中的对应文件。

**Q: 图片尺寸不合适怎么办？**
A: 可以使用在线工具（如Canva、Photoshop）调整图片尺寸。

**Q: 网站没有更新图片怎么办？**
A: 按Ctrl+F5强制刷新浏览器缓存。
